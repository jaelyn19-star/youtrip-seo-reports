📋 Recommended Manual Checks
GSC Checks (do this first)
  • 🔲
    GSC → Core Web Vitals → MobileCheck the Mobile tab for "Poor URLs" and "Needs Improvement" counts. This is real-user field data — more reliable than lab data.
  • 🔲
    GSC → Core Web Vitals → DesktopCompare mobile vs desktop. Large gaps suggest mobile-specific issues (unoptimised images, layout shifts on smaller viewports).
  • 🔲
    Check "URL groups" in the CWV reportGSC groups similar URLs. Blog/article pages are the highest risk — confirm they're not all failing LCP.
  • 🔲
    Mobile Usability ReportAll inspected pages returned VERDICT_UNSPECIFIED via API. Verify in GSC → Mobile Usability that there are no flagged issues.
PageSpeed Checks (spot test key pages)
  • 🔲
    Run PageSpeed on the Homepagepagespeed.web.dev — test both Mobile and Desktop. The homepage is the highest-traffic page and often benchmarks the CMS overall.
  • 🔲
    Test a high-traffic blog pageUse the Alipay China Guide or Japan ATM Guide — these are image-heavy content pages most likely to fail LCP on mobile.
  • 🔲
    Check LCP element on mobilePageSpeed identifies the LCP element. If it's a hero image — ensure it's loading with fetchpriority="high" and is properly compressed (WebP, <100KB).
  • 🔲
    Audit layout shift sourcesIn Chrome DevTools → Performance tab → look for layout shift markers. Common culprits: ad slots, social embeds, fonts loading late.
🔧 Common WordPress CWV Issues to Rule Out
Issue Affects Typical Fix Priority
Hero image not preloaded LCP Add fetchpriority="high" to above-fold img tag High
Images not in WebP format LCP Convert with Imagify/ShortPixel plugin or via CDN transform High
Render-blocking scripts (GTM, ads) LCP, INP Defer non-critical JS; load GTM asynchronously High
Ads or embeds without reserved space CLS Set explicit width/height on ad containers and iframes Medium
Late-loading fonts causing FOUT CLS Use font-display: swap and preload critical fonts Medium
Heavy WordPress plugins (page builders) INP, LCP Audit active plugins; disable on pages where not needed Medium
No caching plugin active TTFB → LCP Enable WP Rocket, W3 Total Cache, or host-level caching Medium
Images missing width/height attributes CLS WordPress auto-adds these for images inserted via media library since WP 5.5 Low
🎯

Target Benchmarks

Google's recommended thresholds for passing Core Web Vitals (based on 75th percentile of page loads):

LCP: <2.5s  ·  CLS: <0.1  ·  INP: <200ms

Content sites on WordPress typically score 40–65/100 on Mobile PageSpeed. Getting to 70+ on Mobile would be a strong result for this site type. Focus on LCP and CLS first — they have the largest correlation with ranking signals.
